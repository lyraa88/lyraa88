  - [**Github Repo**](https://github.com/GDSC-SWU/2024-Moodinary-HultPrize.git)

  - 회귀 모델과 시간 민감형 특징을 적용하여 하이프 트렌드, 발매 시점, 브랜드 효과를 포착했습니다.
